# JavaScript Mini Projects

Three interactive demos covering core JavaScript concepts.

## Projects

### 1. Counter App (`counter/index.html`)
- DOM manipulation with `getElementById`
- Event listeners (`addEventListener`)
- Variables (`let`, `const`), increment/decrement operators

### 2. To-Do List (`todo/index.html`)
- **Closures** — event handlers inside `forEach` close over each task object
- **Scope** — outer `tasks` array accessible inside inner functions
- Dynamic DOM creation (`createElement`, `appendChild`, `innerHTML`)
- Array methods: `push`, `filter`, `splice`, `indexOf`

### 3. Quiz App (`quiz/index.html`)
- **Promises** — `loadQuestions()` returns a `new Promise(...)`
- **async/await** — `startQuiz()` uses `await` to wait for the Promise
- **Hoisting** — function declarations are hoisted (called before defined)
- DOM rendering, event delegation, state management

## Concepts Covered

| Concept | Where |
|---------|-------|
| DOM Manipulation | All 3 |
| Event Handling | All 3 |
| Closures | To-Do List |
| Scope | To-Do List, Quiz |
| Hoisting | Quiz |
| Promises | Quiz |
| async/await | Quiz |

## How to Run

Just open any `index.html` file in your browser — no build tools needed!
